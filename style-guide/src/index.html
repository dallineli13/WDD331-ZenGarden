Dark Garden Style Guide

Purpose/Theme

We decided on going with a "Night Garden" theme for our Zen Garden project. The inspiration for this idea came from the growing popularity of using dark mode on tablets, phones and desktops. The theme should remind users of dark mode by inverting the background and font colors from light to dark. This dark theme will create a strong contrast that should delight visitors with a modern spin to the Zen Garden. We plan to add some variety by incorporating different colored background and fonts while also keeping the dark theme consistent throughout.
